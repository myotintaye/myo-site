GPS Tracker(Mobile Application)-Android,iOS

Description

A cross platform mobile app to monitor an AI-enabled tracking device with GPS/Cellular Data to track and locate children.

Features
  • This application allows parents to track the location of their children using a small GPS device.
  • The device transmits geo-coordinates to the server application which communicates this information to a mobile app.
  • Parents can use the mobile app to receive information about child’s location and alerts if something is out of the ordinary.
  • We use two ways to monitor child’s location, geofences to create a circular area, or fence, around the location of interest, and machine learning, to predict next child’s location based on location history and daily activities
